### Phase 7 — Hardening (this session) ✅

#### 7.1 Agent area passphrase gate

`/agent/` is now publicly reachable but **not** functional without the passphrase:
- Full-screen gate overlay shown on initial visit and after any browser refresh (sessionStorage-scoped)
- Passphrase verified via SHA-256 (Web Crypto API), constant-time hash equality
- Hash stored in source as `AGENT_KEY_HASH` constant (rotateable via documented PowerShell one-liner)
- Different passphrase from atelier — separation of concerns
- "Verrouiller la session" button at top right of dashboard wipes sessionStorage and reloads
- URL hash `#unlock` auto-focuses the input

#### 7.2 XSS hardening

Every dynamic string interpolated into `innerHTML` in `agent.astro` is now passed through a `safe()` helper that HTML-escapes `<>&"'`. Affected render functions:
- `renderProperties` (title, city, id)
- `renderClients` (name, email, phone, id)
- `renderAppointments` (date, note, linked property title, id)
- `viewSubscribers` (email list items)

Same `safe()` pattern already in place on `atelier/EventLog.astro` and `contact.astro` form persistence.

#### 7.3 LocalStorage schema versioning

Added to `src/lib/data.ts`:
- `IMMOPRO_SCHEMA_VERSION = 2` sentinel
- `ensureSchemaCompatible()` runs on every `getProperties()` call
- On version mismatch: wipes `immopro_properties`, `immopro_clients`, `immopro_appointments` (forces re-seed from `INITIAL_PROPERTIES` with the new shape); **preserves** user-only stores (favorites, subscribers, contact messages)
- Bumped from v1 to v2 because we added `neighborhood`/`neighborhoodEn`

- **Remotion `@remotion/player` Astro embed** in the property detail film modal and agent videos tab — placeholders are in place, just need the React island wiring (`npx astro add react`)
- **Remotion Lambda cloud rendering** for MP4 exports
- **Playwright smoke tests** + **Vitest unit tests** for `interpolate`, `clamp`, `t()`, `getProperties`/`saveProperties`
- **Real agent backend** (currently sessionStorage-gated client-side check — adequate for V1, not for production sensitive data)
- **Lighthouse pass** + cross-device manual QA at 375/768/1024/1440 — should be done before any public deploy

### Rotate passphrases before deploy
```powershell
[System.BitConverter]::ToString(
  [System.Security.Cryptography.SHA256]::Create().ComputeHash(
    [System.Text.Encoding]::UTF8.GetBytes("YOUR-NEW-PASSPHRASE")
  )
).Replace("-","").ToLower()
```
- Atelier: replace `KEY_HASH` in `immoPro/src/pages/atelier/index.astro`
- Agent: replace `AGENT_KEY_HASH` in `immoPro/src/pages/agent.astro`
